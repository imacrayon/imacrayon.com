Build a Live "Now Playing" Widget

08 Feb 2023

Here's how I build the "Now Playing" widget on the homepage of this website:

I play most of my music through Spotify. However the Spotify API requires you to setup an OAuth flow and manage authenticaion tokens. Instead, I created a Last.fm account and setup Spotify Scrobbling on the Last.fm Settings page. Authenticating with the Last.fm API is very simple, create an API application, and then include the API Key on the API Applications page along with the API request in the next step.
